# Notification Settings

Notification settings refers to the action of preventing someone from communicating with you. On this page, we'll dive into the different notification settings endpoints you can use to manage notification settings programmatically. We'll look at how to query, upload, update, and delete notification settings. {{ className: 'lead' }}

## List all notification settings {{ tag: 'GET', label: '/v1/notificationsettings' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your notification settings (in a notification settings if a notification settings id is provided). By default, a maximum of ten notification settings are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="limit" type="integer">
        Limit the number of notification settings returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/notificationsettings">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.protocol.notification/v1/notification settings \
      -H "Authorization: Bearer {token}" \
      -d notification settings_id="xgQQXg3hrtjh7AvZ" \
      -d limit=10
    ```

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.notificationsettings.list()
    ```

    ```python
    from protocol_api import ApiClient

    client = ApiClient(token)

    client.notification settings.list()
    ```

    ```php
    $client = new \Protocol\ApiClient($token);

    $client->notification settings->list();
    ```

## Update a notification settings {{ tag: 'PUT', label: '/v1/notificationsettings/:id' }}

<Row>
  <Col>

    This endpoint allows you to perform an update on a notification settings. Examples of updates are pinning a message, muting or archiving the notification settings, or pinning the notification settings itself.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="PUT" label="/v1/notificationsettings/xgQQXg3hrtjh7AvZ">

    ```bash {{ title: 'cURL' }}
    curl -X PUT https://api.protocol.chat/v1/notificationsettings/xgQQXg3hrtjh7AvZ \
      -H "Authorization: Bearer {token}" \
      -d 'is_muted'=true
    ```

    ```js
    import ApiClient from '@example/protocol-api'

    const client = new ApiClient(token)

    await client.notificationsettings.update('xgQQXg3hrtjh7AvZ', {
      notification settingsal_tone: true,
    })
    ```

    ```python
    from protocol_api import ApiClient

    client = ApiClient(token)

    client.notificationsettings.update("xgQQXg3hrtjh7AvZ", is_muted=True)
    ```

    ```php
    $client = new \Protocol\ApiClient($token);
    ```
